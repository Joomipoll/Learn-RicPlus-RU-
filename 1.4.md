# Операции сравнения

Для примера введём две константы:
```C
const int x = 116;
const int y = 404;
```

В языке программирования RicPlus есть следующие операции сравнения:
- "=" Равно
```C
bool a = {x = y};   // false
```
- "<" Меньше
```C
bool b = {x < y};   // true
```
- ">" Больше
```C
bool c = {x > y};   // false
```
- "<=" Меньшее или равно
```C
bool d = {x <= y};  // true
```
- ">=" Больше или равно
```C
bool e = {x >= y};  // false
```
- "!=" Не равно
```C
bool f = {x != y};  // true
```
